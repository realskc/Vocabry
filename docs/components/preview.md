# 预览器设计

## 目标

以固定的参考样式渲染主应用当前交付的 HTML，帮助检查内容结构和正反面，不追求与各版本 Anki WebView 像素级一致。

## 形态

`vocabd` 托管页面与同源 API；`vocab preview` 只负责生成短时访问能力并打开浏览器。页面支持：

- 按 ID 打开或浏览卡片列表；
- 正面/背面翻转；
- 键盘切换；
- 显示卡型、revision、HTML 来源和陈旧状态等调试信息。

第一版只读，不编辑卡片。页面不启用外部脚本、远程资源或 CORS。

## 安全

浏览器 URL 使用短时、限定预览权限的 session，不暴露管理 token。HTML 放在受控容器中；第一版结构化文本生成的 HTML无脚本。Anki 手工 HTML 回写后可能包含任意标记，因此预览必须使用 sandboxed iframe，并禁止脚本、表单、顶层导航和任意源访问。
